## A CLI tool to apply launch options for all Steam games at once.
### Usage:
```
Usage: SteamGlobalLaunchOptions <value> [flags]

A CLI tool to apply launch options for all Steam games at once.

Arguments:
  <value>

Flags:
  -h, --help                    Show context-sensitive help.
  -U, --user-id=""
  -O, --overrite
  -R, --restore-steam

SteamGlobalLaunchOptions: error: expected "<value>"
```
